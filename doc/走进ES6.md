## ES6

ES6 是 2015 年 6 月发布的一个 JavaScript 标准，目的是让 JavaScript 可以开发大型应用，后因决策决定，按年份来命名新的 JavaScript 标准，所以 ES6 更名为 ES2015。

_我们以后的开发中都会使用 ES6 的语法，所以 ES6 规范是我们必须掌握的内容_

知识点清单：

- let 和 const
- 解构语法
- Promise
- Generator 和 Async
- Class
- ...
